主观题一：

请按要求完成index.html（页面标题为“网易新闻“）和index.CSS的所有代码：

有一则图片新闻，效果如上，要求如下：

总体：左图右文，总宽度未知，图文之间间距15px
左图：图片地址假设为http://163.com/x.jpg，图片宽高均为100px

主观题二

已知登录表单效果图如下：

要求如下：

总体：表单提交地址假设为“/login”
标题：“登录网易通行证”为14px的微软雅黑字体，加粗，行高16px，距离第一个输入框15px；
左边文字：“用户名：”和“密码：”所在列列宽60px，文字为12px的宋体，文字与输入框垂直居中对齐；
输入框：输入框宽200px高20px（含边框），边框粗细为1px，颜色为#ddd，两个输入框以及按钮之间间距为15px
按钮：按钮与输入框左对齐，按钮宽40px高20px，蓝底白字无边框，按钮文字为12px的宋体且水平垂直都居中

解答：

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8" />
        <title>登录表单</title>
        <style type="text/css">   
            *{margin:0;padding:0;border:0;}
            form{width:260px;margin:20px auto;}
            .signin legend{margin-bottom:15px;}
            .signin legend span{
                font:bold 14px/16px "Microsoft YaHei";  
                font-weight:bold;               
            }           
            .signin div{
                overflow:hidden;
                margin-bottom:15px 0;               
            }
            .signin div span{
                display:inline-block;
                width:60px;
                font:12px/20px "宋体"; 
                height:20px;                
            }
            .signin div input{
                box-sizing:border-box;
                width:200px;
                height:20px;
                border:1px solid #ddd;
                margin-bottom:15px;
                float:right;
                line-height:20px;
            }           
            .signin div input[type=submit]{
                float:left;
                margin-left:60px;               
                width:40px;
                height:20px;
                background:blue;
                color:white;
                border:0;
                font:12px "宋体";
                line-height:20px;
            }
        </style>        
    </head>

    <body>
        <form action="/login" method="post" class="signin">
        <fieldset>
            <legend><span>登录网易通行证</span></legend>
            <div>
                <label for="user_name"><span>用户名:</span><label>
                <input type="text" id="user_name" name="myName"/>
            </div>
            <div>
                <label for="uesr_password"><span>密码:</span><label>
                <input type="password" id="uesr_password" name="myPassword"/>
            </div>
            <div>
                <input type="submit" value="保存"  id="btn" />
            </div>
        </fieldset>
        </form>
    </body>
</html>

实现效果：

HTML和CSS考点：

  • HTML表单元素语义

  • 表单提交地址设置，标题设置，文字定位设置，输入框设置，按钮设置

  • 左右排列等设置
